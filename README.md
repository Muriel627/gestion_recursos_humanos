# Data Automation 

## Contenido del Repositorio

### Gestión de Recursos Humanos y Presupuesto Universitario
**Core:** Python + SQL Server + Power BI.
- **Automatización:** Script en Python para generación de datos sintéticos y simulación de errores de ERP.
- **Infraestructura:** Modelo de datos relacional en SQL Server (Staging & Data Warehouse).
- **BI:** Dashboard interactivo con DAX avanzado, Row Level Security (RLS) y alertas automáticas.
